# Atlas 🚀

## Internet 100% Décentralisé pour Vidéos, Photos, Textes & Messages

**Atlas** est maintenant un réseau social et de partage multimédia **entièrement P2P et décentralisé** :

### Fonctionnalités
- 📸 **Photos & Vidéos** : Upload direct sur IPFS + partage via Nostr (CID immuable)
- 💬 **Messages & Textes** : Nostr protocol (Kind 1, DMs, feeds)
- 🔄 **100% Décentralisé** : Pas de serveur central – tu choisis tes relays IPFS + Nostr
- 📱 **Toutes plateformes** : iOS • Android • macOS Silicon • Windows • Linux (Flutter)

### Comment lancer (5 min)
1. `flutter create .` (si besoin) puis `flutter pub get`
2. Installe **IPFS Desktop** (kubo) + **Nostr** client comme Damus/Amethyst (mais Atlas remplace)
3. `flutter run`

**Tout est modifiable par moi – dis "ajoute X" et je push direct !**
